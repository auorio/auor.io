Comprehensive APIs

travel_explore Tavily

Real-time web search for AI Agents, with extraction, crawling and mapping to come.

contacts Clado

LinkedIn profile scraping. With contact enrichment APIs soon to come.

map Google Maps

Geolocation information, search for places and detailed information about places.

currency_exchange AllRatesToday

Current and historical exchange rate information.

translate Google Translate

Translation and language detection in many languages.

wb_sunny Google Weather

Current, future, and historical weather information.

beach_access OpenHolidays API

Information about public- and school holidays.

dns IPStack

Information about IP addresses.

Looking for something else? Request a new API to be added.
